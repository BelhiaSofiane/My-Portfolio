Tenzies Game

Tenzies Game is a fun and interactive dice game built with React. The goal is to roll until all ten dice show the same number. Players can click to "hold" dice between rolls, and the game tracks the number of rolls and time taken. A simple yet engaging project that demonstrates React state management and conditional rendering.

Budgeting App is a single-page application built with React and Vite that allows users to track their income and expenses.
